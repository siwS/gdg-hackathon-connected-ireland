# README

Our voludeering app for GDG Connect Ireland Dev Fest!


To run, you need rails-5 and postgresql db

1. clone this repo
2. bundle install
3. setup database preferences
4. rails server


Setup Heroku:

git remote heroku to: https://github.com/siwS/gdg-hackathon-connected-ireland 
git push heroku
